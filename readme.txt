Simple add Google Analytics or Google Tag Manager (GTM) code on your site + Events for Contact Form 7 plugin

== Description ==
Just install and use :) 
Simple Google Tag Manager (GTM) or Universal Google Analytics plugin.
After added Google Analytics ID - select where show your code in Head or Footer section.
Add Events for Contact Form 7 plugin
Add Events for [VDZ CallBack Plugin](https://wordpress.org/plugins/vdz-call-back/)
